# 🚀 SDLC Copilot Backend

An AI-powered backend service that converts raw software requirements into structured development artifacts such as user stories, acceptance criteria, test cases, and ambiguity detection.

---

## 📌 Features

- ✅ Requirement processing using AI
- ✅ Generates:
    - User Stories
    - Acceptance Criteria
    - Test Cases
- ✅ Detects ambiguities in requirements
- ✅ REST API-based architecture
- ✅ Supports multiple AI providers (Gemini / OpenAI fallback)

---

## 🛠 Tech Stack

- Java 17
- Spring Boot
- LangChain4j
- Google Gemini API / OpenAI API
- Maven
